LAB RECORD: Shell Scripting Basics

**Aim:**
To study bash syntax, variables, and control constructs such as if, for, and while loops in shell scripting.

**Theory/Introduction:**
Shell scripting involves writing a series of commands for the Unix/Linux shell to execute. Bash (Bourne Again SHell) is a popular command-line interpreter and scripting language. Shell scripts are powerful tools for automating repetitive tasks, performing system administration, and combining various utilities.

Key concepts covered in this experiment include:
1.  **Bash Syntax:** Understanding the basic rules for writing shell scripts, including comments, executable permissions, and shebang line.
2.  **Variables:** Learning how to declare, assign, and access variables to store data within a script. Shell variables can be local or environment variables.
3.  **Control Constructs:**
    *   **If-Else Statements:** For conditional execution of code blocks.
    *   **For Loops:** For iterating over a list of items or a range of numbers.
    *   **While Loops:** For repeatedly executing a block of code as long as a condition remains true.

**Procedure:**
1.  **Open a Terminal:** Access your Linux/Unix command-line interface.
2.  **Create a Script File:** Use a text editor (e.g., `nano`, `vi`, `gedit`) to create a new file with a `.sh` extension (e.g., `myscript.sh`).
3.  **Add Shebang and Comments:** Start the script with `#!/bin/bash` (shebang) to specify the interpreter and add comments using `#`.
4.  **Write Script Code:** Implement examples for variables, `if-else`, `for`, and `while` loops as detailed in the "Examples" section.
5.  **Save the File:** Save your changes.
6.  **Grant Execute Permissions:** Make the script executable using `chmod +x myscript.sh`.
7.  **Run the Script:** Execute the script from the terminal using `./myscript.sh`.
8.  **Observe Output:** Analyze the output generated by the script to understand the behavior of the commands and control structures.

**Examples:**

**1. Bash Variables:**
```bash
#!/bin/bash
# Script to demonstrate bash variables

# Declaring and assigning variables
NAME="John Doe"
AGE=30
CITY="New York"

# Accessing variables
echo "Name: $NAME"
echo "Age: $AGE"
echo "City: $CITY"

# Reassigning a variable
NAME="Jane Smith"
echo "New Name: $NAME"

# Using command substitution
CURRENT_DIR=$(pwd)
echo "Current Directory: $CURRENT_DIR"
```
**Output:**
```
Name: John Doe
Age: 30
City: New York
New Name: Jane Smith
Current Directory: /path/to/your/directory
```

**2. If-Else Statement:**
```bash
#!/bin/bash
# Script to demonstrate if-else control construct

echo "Enter a number:"
read NUM

if [ "$NUM" -gt 10 ]; then
  echo "$NUM is greater than 10."
elif [ "$NUM" -eq 10 ]; then
  echo "$NUM is equal to 10."
else
  echo "$NUM is less than 10."
fi
```
**Output (Example 1: User enters 15):**
```
Enter a number:
15
15 is greater than 10.
```
**Output (Example 2: User enters 10):**
```
Enter a number:
10
10 is equal to 10.
```
**Output (Example 3: User enters 5):**
```
Enter a number:
5
5 is less than 10.
```

**3. For Loop (Iterating over a list):**
```bash
#!/bin/bash
# Script to demonstrate for loop (list iteration)

FRUITS=("Apple" "Banana" "Cherry" "Date")

echo "Listing fruits:"
for FRUIT in "${FRUITS[@]}"; do
  echo "$FRUIT"
done
```
**Output:**
```
Listing fruits:
Apple
Banana
Cherry
Date
```

**4. For Loop (Numeric Range):**
```bash
#!/bin/bash
# Script to demonstrate for loop (numeric range)

echo "Counting from 1 to 5:"
for i in {1..5}; do
  echo "Count: $i"
done
```
**Output:**
```
Counting from 1 to 5:
Count: 1
Count: 2
Count: 3
Count: 4
Count: 5
```

**5. While Loop:**
```bash
#!/bin/bash
# Script to demonstrate while loop

COUNT=1

echo "Counting up to 3 with while loop:"
while [ $COUNT -le 3 ]; do
  echo "Current count: $COUNT"
  ((COUNT++)) # Increment COUNT
done
```
**Output:**
```
Counting up to 3 with while loop:
Current count: 1
Current count: 2
Current count: 3
```

These foundational elements are crucial for writing efficient and automated shell scripts.
